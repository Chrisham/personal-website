## Getting Started

Simply open `index.html` in a web browser to view the website locally.

For development with live reload, you can use any static server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server
npx http-server

# Using PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.

## Customization

To customize the website for your own use:

1. Update the name and title in `index.html`
2. Modify the content in each section (About, Skills, Projects, Contact)
3. Update contact links and social media URLs
4. Adjust colors in `styles.css` by modifying CSS variables in `:root`
5. Replace project information with your own projects
